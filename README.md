# 🌾 AGRI 2.0

## Overview

AGRI 2.0 is a modern B2B/B2C platform that digitalizes and automates agricultural business workflows including order management, delivery processing, stock tracking, and monthly invoicing.

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js (Express)  
- **Database:** Oracle Autonomous Database  
- **Cloud:** Oracle Cloud Infrastructure (OCI)  
- **Deployment:** Nginx  

## Key Features

- Multi-role platform (Client, Agent, Supplier, Accounting, Director)
- End-to-end digital workflows (Order → Delivery → Invoice)
- Automated document generation
- Stock management per agency
- Real-time dashboards and analytics

AGRI 2.0 improves operational efficiency, ensures data consistency, and supports better decision-making.
